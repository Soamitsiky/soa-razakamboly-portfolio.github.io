# Portfolio — Soa Razakamboly

## 🚀 Installation

### Prérequis
- Node.js (version 16 ou supérieure) : https://nodejs.org

### Étapes

1. **Dézippe** le dossier `portfolio-soa`

2. **Ouvre un terminal** dans ce dossier et lance :
```bash
npm install
npm start
```

3. **Le portfolio s'ouvre** automatiquement sur http://localhost:3000

---

## 📁 Fichiers à ajouter dans `public/`

Place ces fichiers dans le dossier `public/` :
- `photosoa.jpg` — ta photo
- `CV-Soa_Razakamboly-fr.pdf` — ton CV français
- `CV-Soa_Razakamboly-ang.pdf` — ton CV anglais

---

## ✏️ Modifier le contenu

Tout le contenu est centralisé dans **`src/data/data.js`**.
Tu peux y modifier :
- Tes informations personnelles
- Tes projets
- Tes expériences
- Tes recommandations
- Ta recherche d'alternance

---

## 🌐 Héberger sur Netlify (gratuit)

1. Lance `npm run build`
2. Va sur https://netlify.com
3. Glisse-dépose le dossier `build/` généré
4. Ton site est en ligne instantanément !
